# Sausage Jumping Game 🌭🐶

## What is this?

This is a simple endless runner game written in **Java**, inspired by the classic game you play in Google Chrome when your internet gives up on life.

## Features

- written in **Java**
- inspired by the **Chrome Dino** game
- you play as a **sausage dog**
- jump over obstacles
- try not to die
- probably more fun than having no internet

## Gameplay

The rules are simple:

1. run
2. jump
3. survive
4. become the ultimate sausage

The longer you stay alive, the better your score.

## Controls

- **Space** / **Up Arrow** - jump

## Tech Stack

- **Java**
- **Swing / AWT**
